Steps to execute the source code:

Requirements:
1. To run this project you should nodejs installed on your device and a code editor as well.

1. Extract the zip/rar file to the desired destination
2. Type command npm run dev and click on the url provided by the local development server.
3. If the project doesn't execute then check that your selected is powershell, if yes then select any other terminal preferrably git bash or cmd.
